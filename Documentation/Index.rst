===========================
TYPO3 Querybuilder Extension
===========================

:Author:    TYPO3 GmbH Team <info@typo3.com>
:License:   GPL-2.0-or-later
:Rendered:  |today|
:Description:
   The querybuilder extension for TYPO3 provides a filter function based on TYPO3s core list module and features.
   Use multiple given operators to filter your data an save queries for numerous usages on different pages.
